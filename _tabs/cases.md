### 某汽车零部件企业 VSM 精益转型

| 项目 | 内容 |
|------|------|
| **行业** | 汽车零部件制造 |
| **规模** | 约 500 人 |
| **周期** | 6 个月 |
| **方法** | 价值流图（VSM）+ TPS |

**核心问题：**
- 交付周期长达 28 天
- 在制品库存居高不下
- 换模时间长，批次混乱

**改善成果：**
- 交付周期缩短 **62%**（28天 → 11天）
- 在制品库存降低 **45%**
- 换模时间减少 **55%**

### 某电子组装企业 5S + 标准化

| 项目 | 内容 |
|------|------|
| **行业** | 电子制造 |
| **规模** | 约 200 人 |
| **周期** | 3 个月 |
| **方法** | 5S + 标准作业 |

**核心问题：**
- 现场混乱，物料找不到
- 作业标准不统一
- 质量波动大

**改善成果：**
- 现场找料时间减少 **70%**
- 一次合格率提升 **12%**
- 空间利用率提升 **30%**

### 某医疗器械企业流程改善
