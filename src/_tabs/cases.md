### 某汽车零部件企业 VSM 精益转型
### VSM-driven Lean Transformation — Automotive Parts Manufacturer

| 项目 / Item | 内容 / Details |
|------|------|
| **行业 / Industry** | 汽车零部件制造 / Automotive Parts |
| **规模 / Scale** | ~500 人 / Employees |
| **周期 / Duration** | 6 个月 / Months |
| **方法 / Method** | VSM + TPS |

**核心问题 / Key Issues:**
- 交付周期长达 28 天 / Lead time: 28 days
- 在制品库存居高不下 / High WIP inventory
- 换模时间长 / Long changeover time

**改善成果 / Results:**
- 交付周期缩短 **62%**（28d → 11d）
- 在制品库存降低 **45%** / WIP reduced by 45%
- 换模时间减少 **55%** / Changeover reduced by 55%

### 某电子组装企业 5S + 标准化
### 5S & Standardization — Electronics Assembly

| 项目 / Item | 内容 / Details |
|------|------|
| **行业 / Industry** | 电子制造 / Electronics |
| **规模 / Scale** | ~200 人 / Employees |
| **周期 / Duration** | 3 个月 / Months |
| **方法 / Method** | 5S + 标准作业 / Standard Work |

**核心问题 / Key Issues:**
- 现场混乱，物料找不到 / Disorganized workplace

- 空间利用率提升 **30%** / Space utilization +30%

### 某医疗器械企业流程改善
### Process Improvement — Medical Device

| 项目 / Item | 内容 / Details |
|------|------|
| **行业 / Industry** | 医疗器械 / Medical Device |
| **规模 / Scale** | 全院 / Hospital-wide |
