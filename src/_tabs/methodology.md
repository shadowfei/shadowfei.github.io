## 精益方法论体系 / Lean Methodology

精益生产（Lean Production）源于丰田生产系统（TPS），核心思想是通过消除浪费来提升价值。在 20 余年实践中，我们融合精益、六西格玛、TOC（约束理论）和数字化技术，形成了以 **FLOW SMART 卓越绩效运营模型** 为核心的独特方法论体系。

---

### FLOW SMART 卓越绩效运营模型

FLOW SMART 是专为中小企业精益数字化转型设计的原创模型，以最低风险、最小投入追求最高回报。

#### 三大战略支柱 / Three Strategic Pillars

| 支柱 / Pillar | 核心思想 / Core Concept |
|------|------|
| **F**ast Delivery — 快速交付 | 价值流优化、单元化生产、均衡化、快速换模 |
| **L**ow Inventory — 低库存 | 拉动系统、看板管理、物料超市、ABCXYZ 分类 |
| **W**aste Reduction — 消除浪费 | 5S、TPM、标准化、七大浪费识别与消除 |

#### 一个目标 / One Objective

通过战略部署（GDP/Hoshin）将公司长期目标逐层分解为部门和个人可执行的指标，实现上下同欲、力出一孔。

#### 四项流动 / Four Flows

| 流动 / Flow | 内涵 / Meaning |
|------|------|
| **目标流动 / Objective Flow** | 战略→目标→指标→行动，层层对齐 |
| **流程流动 / Process Flow** | 价值流分析→单元设计→均衡生产→连续流 |
| **信息流动 / Information Flow** | 计划→执行→反馈→调整，数据驱动决策 |
| **价值流动 / Value Flow** | 客户需求→订单交付→售后服务，端到端价值交付 |

#### 五项原则 / Five Principles

| 原则 / Principle | 描述 / Description |
|------|------|
| **S**implified — 工具简单化 | 易于操作，一线员工可用，不追求复杂工具 |
| **M**easurable — 绩效数字化 | 数据驱动，实时可视，以事实代替经验 |
| **A**gile — 组织敏捷化 | 快速响应市场变化，柔性调整生产与供应链 |
| **R**eliable — 流程可靠化 | 标准化作业、防错机制、过程可控、结果可预测 |
| **T**ech-Application — 技术应用化 | 低代码平台赋能业务人员，精益牵引数字化 |

> FLOW SMART 以"精益为体，数字化为用"，通过管理体系 + 低代码平台的有机结合，实现"1/10 的成本，10 倍的效益"。

### TPS / Toyota Production System

丰田生产系统（Toyota Production System）是精益生产的起源，由大野耐一创立。

- **准时化 / JIT** — 只在需要的时候，生产需要的数量
- **自働化 / Jidoka** — 带人字旁的自动化，异常即停
- **持续改善 / Kaizen** — 全员参与的渐进式改进

### 价值流图 / Value Stream Mapping

价值流图（VSM）是精益分析的核心工具：

- **现状图绘制 / Current State** — 可视化当前流程，识别浪费（关注流程效率与交付周期）
- **未来图设计 / Future State** — 设计理想流程，消除非增值环节
- **改善计划制定 / Action Plan** — 从现状到未来的路线图

### 5S 现场管理 / 5S Workplace Management

5S 是现场管理的基石：

| 步骤 / Step | 含义 / Meaning | 核心要点 / Key Points |
|------|------|----------|
| 整理 Sort | 区分要与不要 | 清理非必需品 |
| 整顿 Set | 物有其位 | 定置定位，一目了然 |
| 清扫 Shine | 清扫点检 | 设备保养，发现问题 |
| 清洁 Standardize | 标准化 | 制度化，可视化 |
| 素养 Sustain | 习惯养成 | 持续遵守，文化沉淀 |

### 六西格玛 / Six Sigma

数据驱动的改善方法论：

- **DMAIC** — 定义 Define / 测量 Measure / 分析 Analyze / 改进 Improve / 控制 Control
- **SPC** — 统计过程控制，用数据管理过程
- **根因分析 / RCA** — 找到问题的真因

### 战略部署 / Strategy Deployment (Hoshin / GDP)

将企业战略转化为可执行的目标体系：

- **长期愿景 → 年度目标** — 3-5 年战略目标分解为年度突破性目标
- **目标逐层分解** — 公司级 → 部门级 → 个人级，上下对齐
- **日常管理（MDI）** — 日、周、月分层回顾，现场走动管理（Gemba Walk）
- **A3 问题解决** — 用一张 A3 纸结构化定义问题、分析根因、制定对策

### 精益数字化 / Lean Digital

精益与数字化的融合：

- **先精益、后数字化** — 流程优化后再用数字化固化，避免将浪费自动化
- **数据驱动改善 / Data-driven Kaizen** — 实时数据取代经验判断
- **低代码平台 / Low-Code Platform** — 业务人员参与系统构建，快速迭代、低成本落地
- **MES/APS 系统设计** — 结合精益思想的制造执行与高级排程系统设计
- **智能运营 / Smart Operations** — 异常预警 + 自动闭环
