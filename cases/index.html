某汽车零部件企业 VSM 精益转型

VSM-driven Lean Transformation — Automotive Parts Manufacturer

项目 / Item内容 / Details
行业 / Industry汽车零部件制造 / Automotive Parts
规模 / Scale~500 人 / Employees
周期 / Duration6 个月 / Months
方法 / MethodVSM + TPS

核心问题 / Key Issues:

  • 交付周期长达 28 天 / Lead time: 28 days
  • 在制品库存居高不下 / High WIP inventory
  • 换模时间长 / Long changeover time

改善成果 / Results:

  • 交付周期缩短 62%（28d → 11d）
  • 在制品库存降低 45% / WIP reduced by 45%
  • 换模时间减少 55% / Changeover reduced by 55%

某电子组装企业 5S + 标准化

5S & Standardization — Electronics Assembly

项目 / Item内容 / Details
行业 / Industry电子制造 / Electronics
规模 / Scale~200 人 / Employees
周期 / Duration3 个月 / Months
方法 / Method5S + 标准作业 / Standard Work

核心问题 / Key Issues:

  • 现场混乱，物料找不到 / Disorganized workplace
  • 作业标准不统一 / Inconsistent work standards
  • 质量波动大 / High quality variation

改善成果 / Results:

  • 现场找料时间减少 70%
  • 一次合格率提升 12% / First-pass yield +12%
  • 空间利用率提升 30% / Space utilization +30%
