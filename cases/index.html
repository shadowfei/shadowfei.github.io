某制药企业 SMED 快速换模

SMED Rapid Changeover — Pharmaceutical Packaging

项目 / Item内容 / Details
行业 / Industry制药包装 / Pharmaceutical Packaging
方法 / MethodSMED（单分钟快速换模）

核心问题 / Key Issues:

  • 压片机换模时间长，影响有效产出
  • 换模操作不规范，质量波动大

解决方案 / Approach: 运用 SMED 方法论将换模步骤区分为内部与外部作业：

  1. 测时与录像分析（773 分钟基线）
  2. 识别内外部作业，将可并行操作转为外部
  3. 优化剩余内部作业，标准化操作流程

改善成果 / Results:

  • 换模时间从 773 分钟降至 355 分钟（缩短 51.5%） / Changeover from 773 to 355 min (-51.5%)
  • 单次换模节省 6+ 小时，折算年产值超 2000 万
